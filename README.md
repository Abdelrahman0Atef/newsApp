News App Flutter
A basic news application using Flutter in which one can view news articles about the US, under different categories.
It utilizes Dio for fetching news data from a REST API and displays content from external websites with the help of WebView.

Features
Fetch news articles from a REST API using Dio.
Select news categories to filter the content, such as Sports, Technology, Business, etc.
Display news articles in a WebView to make reading easy inside the app itself.
simple and intuitive interface for navigation through news articles.
